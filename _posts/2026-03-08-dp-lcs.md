## 정의

두 수열 X, Y가 있을 때, 둘 다의 부분 수열이면서 가장 긴 것을 LCS라 한다.

## DP 점화식

```
dp[i][j] = X[i] == Y[j] ? dp[i-1][j-1] + 1
                         : max(dp[i-1][j], dp[i][j-1])
```

## 구현

```cpp
int n, m;
string x, y;
int dp[1001][1001];

for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++)
        if (x[i-1] == y[j-1]) dp[i][j] = dp[i-1][j-1] + 1;
        else dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
```
